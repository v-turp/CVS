This app uses a FLickr API to display images related to a given searchbar search.
Code is written in Kotlin, uses Jetpack Compose and MVVM architecture. Kotlin Flows
are utilized in the app and the apps unit tests.

HOW THE APP WORKS:
The search results are updated after each keystroke or change to the search string.
• When performing the search, a progress indicator is displayed. 
• Tapping on an image display an image detail view that contains the following details:
o Text element that displays the title
